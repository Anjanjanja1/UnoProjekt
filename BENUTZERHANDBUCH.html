Einleitung

Willkommen zu unserem UNO Spiel! Diese Anleitung hilft Ihnen, die Regeln, die Spielvorbereitung und die Spielmechanik des Spieles der Gruppe 4 zu verstehen. Befolgen Sie die Schritte und Anweisungen sorgfältig, um ein reibungsloses und angenehmes Spielerlebnis zu gewährleisten.

Spielvorbereitung

Um das Spiel zu starten, benötigen Sie 2-4 Spieler. Sie können zwischen menschlichen Spielern oder Bots wählen. Jeder Spieler kann seinen Namen eingeben. Das Kartendeck wird gemischt und jeder Spieler erhält 7 zufällige Karten. Die restlichen Karten bilden den Nachziehstapel, und die oberste Karte wird aufgedeckt und auf den Ablagestapel gelegt. Der erste Spieler beginnt die Runde.

Spielablauf

Das Spiel überprüft die oberste Karte auf dem Ablagestapel. Wenn es eine Aussetzen- oder Richtungswechselkarte ist, wird der aktuelle Spieler übersprungen oder der Spielablauf ändert seine Richtung. Wenn es eine +4 Karte ist, zieht der aktuelle Spieler automatisch 4 Karten. Wenn es eine +2 Karte ist, kann der Spieler eine weitere +2 Karte spielen, um das Ziehen von Karten zu vermeiden, bis ein Spieler keine +2 Karte mehr hat und die angesammelten Karten ziehen muss.

Die Spieler können eine Karte ziehen, eine Karte spielen, UNO sagen oder zum nächsten Spieler wechseln, indem sie den entsprechenden Befehl eingeben. Wenn ein Spieler versucht, zweimal in einem Zug zu spielen, erhält er eine Strafkarte. Spieler müssen "UNO" sagen, wenn sie nur noch eine Karte haben. Wenn dies nicht erfolgt, werden automatisch zwei Strafkarten gezogen.

Um eine Karte zu spielen, geben Sie den Indexwert Ihrer möglichen Karten ein. Zum Beispiel, wenn Ihre Karten [B3, G5, Y2, R6] sind und Sie G5 spielen möchten, geben Sie den entsprechenden Indexwert für G5 ein, der in dem Falle "1" ist.

Die gezogene Karte kann sofort gespielt werden, wenn sie zur obersten Karte passt; andernfalls wechselt der Spieler zum nächsten Zug.

Aktionskarten

PLUS ZWEI KARTE

Der Spieler muss 2 Karten ziehen. Diese Karte kann auf eine Karte derselben Farbe oder eine andere +2 Karte gelegt werden. Sie kann nicht als erste Karte gespielt werden.

RICHTUNGSWECHSEL KARTE

Ändert die Spielrichtung. Diese Karte kann auf eine Karte derselben Farbe oder eine andere Richtungswechselkarte gelegt werden. Sie kann die erste Karte des Spieldecks sein..

AUSSETZEN KARTE

Überspringt den nächsten Spieler. Diese Karte kann auf eine Karte derselben Farbe oder eine andere Aussetzen-Karte gelegt werden. Sie kann die erste Karte des Spieldecks sein.

WILDE KARTE

Ändert die Farbe, die als nächstes gespielt werden soll. Diese Karte kann jederzeit auf jede Karte gelegt werden. Sie darf nicht die erste Karte des Spieldecks sein.

WILDE +4 KARTE

Ändert die Farbe, die als nächstes gespielt werden soll, und der nächste Spieler zieht automatisch 4 Karten. Der Spieler muss einen Zug mit der gewählten Farbe oder einer Spezialkarte machen. Diese Karte kann jederzeit auf jede Karte gelegt werden. Sie darf nicht die erste Karte des Spieldecks sein.

Strafen

Jeder Spieler muss eine Karte ziehen oder spielen. Wenn er das nicht tut, erhält er eine Strafkarte.

Wenn vergessen wird "UNO!" zu sagen, oder es gesagt wird, obwohl man mehr als eine Karte in der Hand hat, führt dazu, dass zwei Strafkarten gezogen werden.

Das Spielen einer falschen Karte führt automatisch dazu, dass der Spieler eine Strafkarte erhält und der nächste Spieler an der Reihe ist.

Punktewertung

Punkte werden basierend auf den Karten vergeben, die sich noch in den Händen der anderen Spieler befinden, wenn ein Spieler die Runde gewonnen hat und keine Karten mehr in der Hand hält:

  • 0-9: Kartenwert
  • +2 Karte: 20 Punkte
  • Richtungswechsel Karte: 20 Punkte
  • Aussetzen Karte: 20 Punkte
  • Wilde Karte: 50 Punkte
  • Wilde +4 Karte: 50 Punkte

Spielende

Der erste Spieler, der alle seine Karten ablegt, erhält Punkte für die restlichen Karten der anderen Spieler. Die Punkte werden angezeigt und in einer Datenbank gespeichert. Das Spiel wiederholt sich, bis ein Spieler 500 Punkte erreicht. Der erste Spieler, der 500 Punkte erreicht, gewinnt das Spiel, und das Spiel endet. Der Gewinner wird am Ende des Spiels bekannt gegeben.
